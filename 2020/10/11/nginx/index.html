Centos V2ray + Nginx + WS

Posted on 2020-10-11 Edited on 2024-04-13

  1. yum -y update

  2. yum -y install gcc pcre pcre-devel zlib zlib-devel openssl openssl-devel

  3. yum -y install nginx

  4. bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)

  5. yum -y install git

  6. git clone https://github.com/letsencrypt/letsencrypt

  7. ./letsencrypt/letsencrypt-auto certonly –standalone -d jadehare.com

  8. 修改nginx配置

  1. mkdir /etc/nginx/log/

  2. touch /etc/nginx/log/jadehare.com_nginx.log

  3. 添加v2ray 配置

touch /etc/v2ray/config.json

  1. nohup v2ray -config /etc/v2ray/config.json 2>&1 &
