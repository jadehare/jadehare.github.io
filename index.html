Centos V2ray + Nginx + WS

Posted on 2020-10-11

1.yum -y update
2.yum -y install gcc pcre pcre-devel zlib zlib-devel openssl openssl-devel
3.yum -y install nginx

4.bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)

5.yum -y install git
6.git clone https://github.com/letsencrypt/letsencrypt
7../letsencrypt-auto certonly –standalone -d yoururl.com

8.修改nginx配置

9.添加v2ray 配置

touch /etc/v2ray/config.json

nohup /usr/local/bin/v2ray -config /etc/v2ray/config.json 2>&1 &

10.BBR加速
wget “https://github.com/cx9208/bbrplus/raw/master/ok_bbrplus_centos.sh" && chmod +x ok_bbrplus_centos.sh && ./ok_bbrplus_centos.sh

11.加速出问题

vim /etc/sysctl.conf

  • 删除或注释掉其中的两行：

    #net.core.default_qdisc = fq 用#注释掉

    #net.ipv4.tcp_congestion_control = bbr 用#注释掉

Cocos Creator Android and IOS Debug

Posted on 2020-08-05 Edited on 2020-08-10

IOS

第一步:构建项目

第二步:Xcode 打开项目 build/jsb-default/frameworks/runtime-src/proj.ios_mac

第三步:使用外部 ios 设备安装项目;

第四步:打开苹果原生 safari 浏览器,打开开发者->开发->模拟器 Iphone8->cocos2d-x jsb;

第五步:找到对应的代码(src/project.js);

Android

第一步:构建项目

第二步:AndroidStudio 打开项目 build/jsb-default/frameworks/runtime-src/proj.android-studio

第三步:使用外部 Android 设备运行项目;

第四步:

  • App 启动后控制台输入 adb logcat|grep “chrome-devtools”;

  • 输出 log 04-17 14:05:51.386 9858 10015 D jswrapper: Debugger listening…, visit [ chrome-devtools://devtools/bundled/inspector.html?v8only=true&ws=0.0.0.0:5086/00010002-0003-4004-8005-000600070008 ] in chrome browser to debug!

  • 控制台输入 adb forward tcp:5086 tcp:5086;

第五步: 浏览器打开：devtools://devtools/bundled/inspector.html?v8only=true&ws=0.0.0.0:5086/00010002-0003-4004-8005-000600070008 开始调试
